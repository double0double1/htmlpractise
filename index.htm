My first website using Python and Django

This work is done as an assignment for the Web Application Technologies and Django course as part of the Django For Everybody Specialization provided by University of Michigan via Coursera website. This assignment is intended to improve the basic knowledge of a student in HTML, by creating a simple web page and hosting it via pythonanywhere.

This is the first time I'm getting my hands on creating a web page using html. The creation of an HTML page is a very satisfying process as it's wonderful to observe how inserting a few tags in between the text creates a well defined structure for the document.

To finish and earn the grades for this assignment, I'm supposed to meet some criteria in the HTML document which Im writing.

The following requirements are required for the completion of the assignment:
  • I should be using at least one example of each of the following tags: span, p, div, h1, and ul
  • I must include three a tags and three li tags.
  • I must have some bold and italics text in the document without using b or i tags.

Although there is a lot of flexibility within those parameters, it can still be a little challenge for a first timer. First look at this html says that I've met those parameters to pass the assignment, I still have to run the HTML content in the W3C validator and then post the link in the course autograder to verify if I've included everything.
